# LIDYA FASHION

A modern fashion e-commerce platform with AI-powered style preview functionality.

## Features

- **AI Style Preview** - See yourself in any outfit before you buy
- **Responsive Design** - Optimized for mobile and desktop
- **Fast Performance** - Built with Next.js 16 and Turbopack
- **Modern UI** - Beautiful components with shadcn/ui and Tailwind CSS

## Tech Stack

- **Next.js 16** - React framework with App Router
- **TypeScript** - Type-safe development
- **Tailwind CSS 4** - Utility-first styling
- **shadcn/ui** - Beautiful UI components
- **Replicate API** - AI image generation

## Getting Started

```bash
# Install dependencies
bun install

# Start development server
bun run dev

# Build for production
bun run build
```
